# Autonomous Agents

AI agent loops that run micro-businesses without human involvement.
Observe -> Decide -> Act -> Earn -> Repeat.

## Strategy
- Build agents that operate 24/7 autonomously
- Each agent runs a self-contained micro-business
- Revenue through arbitrage, sales, commissions, or ads

- [ ] Print-on-demand agent (generate designs, upload, test, scale winners)
- [ ] Data aggregation agent (scrape, structure, sell via API)
- [ ] Deal finder agent (monitor markets, find underpriced assets)
