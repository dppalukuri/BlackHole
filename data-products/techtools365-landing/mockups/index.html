TechTools365 · Landing Page Mockups

Four minimal variations + three other directions. All use the same content.

Minimal — four variations

1a. Dark, mesh gradient original

Dark bg with subtle radial mesh, Inter Tight, SVG icons, mono captions, fine grain. Quiet confidence with a glow.

1b. Light, one accent new

Off-white paper, black type, single coral accent used sparingly. Generous whitespace, line-based stats, cards separated by hairline rules. Like Linear docs.

1c. Swiss grid new

12-col grid visible as hairlines, oversized lowercase headline, vertical rules between cells, products listed editorially with numbered index. Bureau-of-graphic-design feel.

1d. Typography-driven new

Dark bg, the hero is the design — 13rem display type stacked 4 lines tall, acid-lime accent, scrolling "no signup · no tracking" marquee, product names as huge typographic entries. Bold, opinionated.

Other directions

2. Illustrated & Playful Notion/Duolingo

Cream background, sticker-style cards with drop shadow, hand-drawn SVG doodles, rounded blob icons, Caveat cursive accents.

3. Data-forward Stripe dashboard

Dashboard layout, sparklines per card, IBM Plex, terminal-green accent — shows traction as proof.

4. Editorial NYT Mag

Cream paper, Fraunces serif display, magazine feature + sidebar grid, numbered stories, SVG cover art.
Picking between the 4 minimal variants — 1a feels tech-startup, 1b feels like careful professional software, 1c feels like a design studio / Swiss publication, 1d feels bold / operator brand. Think about which one would also look right headlining CalcStack (financial tool) and VisaPathway (travel tool).
